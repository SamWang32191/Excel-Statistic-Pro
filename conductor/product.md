# Initial Concept
這是一個專為個案管理與數據統計設計的現代化 Web 工具。使用者可以上傳 XLSX 檔案，自動統計 個案名單 或 服務清冊，並將結果一鍵同步或累加至 Google Sheets。

# 產品願景
打造一個高效且精準的數據處理工具，消除行政人員在手動統計與上傳數據時的負擔，確保數據的一致性與即時性。

# 目標對象
- **行政人員**：日常需處理複雜 Excel 報表，並將結果彙整至雲端試算表的人員。

# 核心目標
- **流程簡化**：實現一鍵式數據讀取、統計與雲端同步，取代繁瑣的人工作業。
- **靈活配置**：適應各種 Excel 格式，提供使用者自定義欄位映射的彈性。
- **自動化管理**：根據時間與行政區劃分，自動管理 Google Sheets 工作表，保持資料井然有序。

# 核心功能
- **雙模式統計引擎**：支援「個案名單」與「服務清冊」兩種核心業務模式，並可根據需求擴展。
- **智能欄位對應**：允許使用者輸入 Excel 列代號，靈活解析不同來源的檔案結構。
- **自動化工作表調度**：依據「縣市 + 月份」自動建立或切換對應的工作表，實現數據分流。
- **增量寫入邏輯**：同步至 Google Sheets 時採用累加模式，保護既有數據不被覆蓋。

# 視覺與互動體驗
- **效率導向介面**：採用簡潔配置，確保數據統計結果清晰易讀，減少視覺干擾。
- **即時反饋**：提供即時統計預覽與上傳進度提示，增加操作的透明度與信心。

# 未來展望
- **擴展統計模式**：持續整合更多業務場景的統計邏輯，擴大工具的適用範圍。
