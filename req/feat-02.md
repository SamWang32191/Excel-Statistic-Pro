| (65歲以上 老人)                                   |      |
| ------------------------------------------------- | ---- |
| 獨居                                              | 97   |

| 與家人住                                          | 244  |
| 與朋友住                                          | 0    |
| 其他                                              | 11   |

| (50~64 身障)                                      |      |
| ------------------------------------------------- | ---- |
| 獨居                                              | 13   |
| 僅配偶2人                                         | 0    |
| 與家人住                                          | 22   |
| 與朋友住                                          | 0    |
| 其他                                              | 2    |
